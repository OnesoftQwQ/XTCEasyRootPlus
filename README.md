<h1 align="center">XTCEasyRootPlus</h1>

<p align="center">一个使用 Python 制作的小天才电话手表一键 ROOT 程序</p>

本项目逻辑基本上参照 XTCEasyRoot 制作,相比 XTCEasyRoot,本项目拥有更阳间的编程语言,更好看的 Ui,更简化的逻辑
